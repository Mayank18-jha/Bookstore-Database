# Online Bookstore Database System

## Problem Statement
Managing books, customers, and orders requires a structured database. This project implements a normalized bookstore database.

## Features
- Book inventory management
- Customer management
- Order processing
- SQL JOIN-based order summary

## Tech Stack
- Python
- SQLite
- SQL

## How to Run
python bookstore.py

## Future Improvements
- Order cancellation
- Stock alerts
- Payment simulation
